    Java 8 – Collectors groupingBy and mapping example

    By mkyong | April 7, 2016 | Viewed : 578 times

    A Java example to show you how to use the Java 8 Collectors.groupingBy and Collectors.mapping

    1.1 A POJO.

    Person.java
    package com.mkyong.java8;public class Person {        private String name;        private String city;        //getters, setters, constructor, toString     }}

    1.2 Java 8 group by and mapping example, it should be self-explanatory.

    Person.java
    package com.mkyong.java8;import java.util.Arrays;import java.util.List;import java.util.Map;import java.util.Set;import java.util.stream.Collectors;public class TestJava8 {    public static void main(String[] args) {        TestJava8 obj = new TestJava8();        obj.test();    }    private void test() {        List<Person> persons = Arrays.asList(                new Person("mkyong", "new york"),                new Person("mkyong", "chicago"),                new Person("mkyong", "new york"));        Map<String, List<Person>> collect = 			persons.stream().collect(Collectors.groupingBy(Person::getCity));        System.out.println(collect);        //multi-level reduction, toSet()        //{chicago=[mkyong], new york=[mkyong]}        Map<String, Set<String>> collect1 = 			persons.stream().collect(Collectors.groupingBy(Person::getCity,                Collectors.mapping(Person::getName, Collectors.toSet())));        System.out.println(collect1);        //multi-level reduction, toList()        //{chicago=[mkyong], new york=[mkyong, mkyong]}        Map<String, List<String>> collect2 = 			persons.stream().collect(Collectors.groupingBy(Person::getCity,                Collectors.mapping(Person::getName, Collectors.toList())));        System.out.println(collect2);    }}

    Output

    {chicago=[Person{name='mkyong', city='chicago'}], 	new york=[Person{name='mkyong', city='new york'}, Person{name='mkyong', city='new york'}]}{chicago=[mkyong], new york=[mkyong]}{chicago=[mkyong], new york=[mkyong, mkyong]}
    Note
    Do you have another use case for Collectors.mapping? Please share it with me, thanks.
